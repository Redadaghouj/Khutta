# خُطّة - طريقك إلى العلم الشرعي

موقع عربي بسيط يهدف إلى مساعدة المسلم على بدء رحلته في طلب العلم الشرعي وحفظ القرآن الكريم من خلال تقديم خارطة طريق واضحة تجمع بين:

* معهد مرتقى لدراسة العلوم الشرعية.
* تطبيق مراقي لحفظ القرآن الكريم ومراجعته مع المشايخ المتخصصين.

يعرض الموقع مساراً تعليمياً للمبتدئين، ودورات مقترحة للانطلاق، بالإضافة إلى خطة منهجية تمتد لثلاث سنوات.
